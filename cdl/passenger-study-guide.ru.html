Примеры вопросов

Во время предрейсового осмотра автобуса работу чего нужно проверить?

  1. Аварийные выходы и фонари ✓ Правильный ответ
  2. Кондиционер в салоне
  3. Раковину в туалете
  4. Лампы для чтения сверху

Почему: Pre-trip the bus and make sure the emergency exits, exit lights, and stop-signal/door work.

Во время предрейсового осмотра, что должно быть свободно от багажа?

  1. Сиденье водителя
  2. Проход и все выходы ✓ Правильный ответ
  3. Крышка топливного бака
  4. Линия для стоящих пассажиров

Почему: During the pre-trip inspection, you must make sure the aisle and all exits are clear of baggage.

Ручную кладь нужно закрепить так, чтобы она не могла перекрыть что?

  1. Сиденье водителя
  2. Кассу для оплаты проезда
  3. Проход или любой выход ✓ Правильный ответ
  4. Стеклоочистители

Почему: Carry-on baggage must be secured so it cannot block the aisle or any exit.

Ручная кладь должна быть закреплена так, чтобы она не могла что сделать?

  1. Загородить обзор водителю
  2. Пересечь линию стоящих
  3. Коснуться пола
  4. Упасть на кого-то ✓ Правильный ответ

Почему: Carry-on baggage must be secured so it cannot fall on someone or block the aisle or exits.

Где должны находиться стоящие пассажиры в автобусе?

  1. За линией для стоящих пассажиров ✓ Правильный ответ
  2. Перед линией для стоящих пассажиров
  3. За сиденьем водителя
  4. Рядом с передней дверью

Почему: Standees must stay behind the standee (white) line so they do not block the driver's view.

Где должны находиться стоящие пассажиры в автобусе?

  1. Перед линией для стоящих
  2. За линией для стоящих ✓ Правильный ответ
  3. Рядом с водителем
  4. В переднем проходе

Почему: Standees must stay behind the standee line and never block the driver's view.
